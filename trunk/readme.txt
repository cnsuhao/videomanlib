SETUP
=====
Download cmake and create the solution for your building system
http://www.cmake.org/

EXTERNAL DEPENDENCIES 
=========
Optional:
-OpenGL: Necessary to build VideoMan OpenGL Renderer
-GLUT: Used in some examples http://www.xmission.com/~nate/glut.html
-QT: Used in some examples http://qt.nokia.com/
-FreeType: Used in some examples http://www.freetype.org/
-DirectShow: Necessary to build the VMDirectShow input
-OpenCV: Used in some examples and necessary to build VMHighgui and VMImageSequence inputs http://sourceforge.net/projects/opencvlibrary/
-PointGrey SDK: Necessary to build the VMPointGrey and VMPointGrey2 inputs http://www.ptgrey.com/
-IDS SDK: Necessary to build the VMIDSFalcon and VMIDSuEye inputs http://www.ids-imaging.de/
-Libdc1394: Necessary to build the VMDC1394 input http://damien.douxchamps.net/ieee1394/libdc1394/

BUILDING
=========

1-Install Cmake http://www.cmake.org.
2-Generate the makefile or workspace for your compiler environment.
3-Build the VideoMan control dll.
4-Build the module inputs, please read the instructions file located in the module input directory for additional configurations.
5-Build and play with the examples.

USING VIDEOMAN
==============
1-Include the files  VideoManControl.h and VideoManInputFormat.h
2-Link the VideoMan.lib
3-Copy the VideoMan.dll and the input module dlls to your working directory
4-The API documentation can be found here
http://videomanlib.sourceforge.net/index.php?option=com_content&task=view&id=37&Itemid=52
5-You can use the config.cmake.in file in order to configure your project using Cmake. This file is automatically generated when the library is built
